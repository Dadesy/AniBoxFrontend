## [1.0.0] - 2026-03-18

### Добавлено
- Реализован MVP frontend для аниме-сервиса.
- Раздельная Docker-конфигурация для production и локальной разработки.
- Бесконечная подгрузка элементов в каталоге, поиске и истории.
- Общая логика определения API URL для браузера и server-side запросов в локальном Docker-режиме.
- Главная страница с hero-блоком, подборками и расписанием.
- Каталог, поиск, история просмотра и страницы тайтлов.

### Изменено
- Главная страница переведена на подборки и расписание, построенные на метаданных.
- Поиск, каталог и главная страница ограничены аниме-контентом.
- Локальный запуск frontend переведён на отдельные `docker-compose.local.override.yml` и команды `Makefile`.

### Исправлено
- Проблемы с резолвом компонентов на главной странице и в других Nuxt-представлениях.
- Ошибки загрузки метаданных и проблемы рендера главной страницы, связанные с кэшированием.
- Несовпадения API URL между SSR и клиентом в локальном Docker-режиме.
- Ошибка рендера фильтров каталога, вызванная пустыми значениями `SelectItem`.
